# 🚗 Car Rent Application

## 🌐 Overview

Car Rent Application është një aplikacion web modern që simulon një platformë për **qira të automjeteve**, i dizajnuar për të ofruar një eksperiencë të thjeshtë, të shpejtë dhe intuitive për përdoruesit.

Ky projekt është ndërtuar si një zgjidhje bazë për menaxhimin e rezervimeve të makinave dhe demonstron praktikat më të mira në zhvillimin front-end dhe përdorimin e Git për versionim.

---

## ✨ Key Features

* 🔍 Shfletimi i makinave të disponueshme
* 📄 Shikimi i detajeve të automjeteve
* 📝 Formë rezervimi e thjeshtë dhe funksionale
* 🎨 Dizajn i pastër dhe responsive
* ⚡ Navigim i shpejtë dhe intuitiv

---

## 🛠️ Tech Stack

Ky projekt është zhvilluar duke përdorur teknologjitë e mëposhtme:

* **HTML5** – strukturë semantike moderne
* **CSS3** – stilim dhe layout responsive
* **JavaScript (Vanilla JS)** – logjikë dhe ndërveprim
